# Pages

This directory contains page-level components (route components).

## Structure

- Each page represents a route in the application
- Pages compose multiple components together
- Include page-specific logic and layouts

## Example

```
pages/
├── HomePage/
│   ├── HomePage.tsx
│   ├── HomePage.module.css
│   └── index.ts
├── AboutPage/
│   ├── AboutPage.tsx
│   └── index.ts
└── index.ts (barrel export)
```
